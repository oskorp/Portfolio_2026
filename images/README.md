# Portfolio Image Directory Guide

## How to Replace Placeholder Images

Simply replace the `.svg` placeholder files with your actual photos.
**Keep the same filenames** (just change the extension to .jpg, .png, etc.)
and update the `src` attributes in `index.html` accordingly.

## Adding More Photos
To add more items to any gallery, copy an existing gallery item block in `index.html` 
and increment the filename number. The grid layout is automatic.

## Caption Customization
Edit the `<span class="gallery-caption">` text inside each gallery item in `index.html`.
